# 코틀린의 기본 요소: 함수와 변수

## 1. 코틀린 함수 소개

코틀린에서 함수는 'fun' 키워드로 시작한다. 함수 선언 시 주요 특징은 다음과 같다.

- 반환 타입을 생략할 수 있다.
- 함수를 클래스 외부에 정의할 수 있다.
- 매개변수 이름 뒤에 타입을 지정한다.

## 2. 코틀린 변수 이해하기

코틀린에서 변수는 'val'과 'var'로 선언한다.

- 'val': 불변(immutable) 변수
- 'var': 가변(mutable) 변수

## 3. 코틀린의 특별한 기능

- 타입 추론: 변수와 함수의 반환 타입을 자동으로 추론한다.
- 문자열 템플릿: 문자열 내에 변수나 표현식을 쉽게 삽입할 수 있다.
